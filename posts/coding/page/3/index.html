2022-08-10 12:02:31 #折腾

具体实现思路是，直接前端解析 sitemap.xml ，随机获取一个 url loc 链接，同时增加了判断，若不小心取到以下链接（首页、标签页、分类页等，采用 .split('/')[3] 取主域名 / 后的字符为判断依据），则再循环取一个。
